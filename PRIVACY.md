## Data Storage and Security

### Storage Location
- **Session Data:** Stored in browser session state (temporary, client-side)
- **Uploaded Files:** Processed in memory, not permanently stored on servers
- **Assessment Results:** Available for download, not retained after session ends

### Security Measures
- Secure API communication with encryption
- No permanent server-side storage of personal data
- Session-based data cleared on browser close or manual reset
- API keys stored securely using Streamlit secrets management

### Data Retention
- **During Session:** Data is retained in browser memory during active assessment
- **After Session:** All session data is cleared when you close the browser or reset the assessment
- **No Long-Term Storage:** We do not maintain a database of candidate information

## Your Rights Under GDPR

As a data subject, you have the following rights:

### 1. Right to Access
You can request access to all personal data we hold about you. Since data is session-based, you can view all collected information during your assessment.

### 2. Right to Rectification
You can correct any inaccurate personal information during the assessment process by using the "Reset Assessment" button and re-entering correct information.

### 3. Right to Erasure (Right to be Forgotten)
You can delete all your data at any time by:
- Clicking the "Reset Assessment" button in the sidebar
- Closing your browser (clears session data)
- Not downloading the assessment report

### 4. Right to Data Portability
You can download your assessment data in JSON or text format at the end of the assessment.

### 5. Right to Object
You can object to data processing at any time by:
- Ending the assessment early
- Not providing certain information (though this may limit assessment functionality)
- Closing the application

### 6. Right to Withdraw Consent
You can withdraw consent at any time by closing the application or resetting the assessment. This will immediately stop all data processing.

## Data Protection Principles

We adhere to the following GDPR principles:

1. **Lawfulness, Fairness, and Transparency:** We process data lawfully with your explicit consent and provide clear information about our practices.

2. **Purpose Limitation:** Data is collected only for the specific purpose of technical assessment and is not used for any other purpose.

3. **Data Minimization:** We collect only the minimum data necessary for conducting the assessment.

4. **Accuracy:** You have the ability to correct information during the assessment process.

5. **Storage Limitation:** Data is stored only for the duration of your session and is not retained afterward.

6. **Integrity and Confidentiality:** We implement appropriate security measures to protect your data.

7. **Accountability:** We take responsibility for compliance with data protection principles.
